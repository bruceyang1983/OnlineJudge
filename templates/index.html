OJ 返回结果

对 OJ 系统的判题返回结果的说明。

  • AC -- Accepted
    你的程序完全正确，通过了 OJ 系统的测试
  • Queuing & Compiling & Running
    正在排队判题、正在编译、正在判题
  • PE -- Presentation Error
    你的程序的运行输出格式跟题目描述的并不严格一致，请仔细审题
  • RE -- Runtime Error
    你的程序在运行的时候发生崩溃。有可能是 数组越界、浮点除零、栈溢出等问题
  • TLE -- Time Limit Exceeded
    你的程序在题目所限时间内没有给出所有答案
  • MLE -- Memory Limit Exceeded
    你的程序在运行的时候尝试使用的内存超出了题目所限
  • CE -- Compilation Error
    你的程序在 OJ 系统中没有编译成功，请检查语法
  • WA -- Wrong Answer
    你的程序的运行结果跟标准结果不完全一样
